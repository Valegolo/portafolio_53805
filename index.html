Desarrollo Web

Como comunicadora social especializada en la creación y desarrollo de páginas web, poseo una amplia experiencia trabajando con diversos CMS, incluyendo AEM, Sitecore, Wordpress y Magnolia.

Mi enfoque se centra en fusionar la comunicación efectiva con la tecnología, utilizando mis habilidades en HTML5, CSS y JavaScript para construir experiencias digitales memorables y funcionales.

Actualmente, me encuentro inmersa en un programa de estudios fullstack, fortaleciendo mis conocimientos para ofrecer soluciones integrales y adaptarme a las últimas tendencias tecnológicas. Mi pasión por la comunicación y el desarrollo web se refleja en cada proyecto que abordo, buscando siempre innovar y crear conexiones significativas en el entorno digital.
